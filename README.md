# libmongocrypt-rust
[![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)](LICENSE)

Rust language bindings for the [libmongocrypt](https://github.com/mongodb/libmongocrypt) library for client-side encryption in MongoDB drivers.

# Requirements

libmongocrypt 1.5.0 must be installed; if it's not in a system-default location `MONGOCRYPT_LIB_DIR` can be set to specify a search location.
